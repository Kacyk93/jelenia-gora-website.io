# 📍 Jelenia Góra — Information Website

A front‑end portfolio project presenting key information about the city of Jelenia Góra, its attractions, articles, and a fully functional contact form with PL/DE language support.

## 🛠️ Technologies

- **HTML5** — semantic structure  
- **CSS3** — variables, grid, responsiveness, animations  
- **JavaScript (ES6)** — translations, parallax, fade animations, hamburger menu  
- **GitHub Pages** — project hosting  

---

## ✨ Features

### 🌍 Multilingual (PL / DE)
- dynamic text translations  
- translated placeholders  
- translated alt and title attributes  
- language stored in localStorage  

### 🎨 Design & UX
- full‑screen hero with parallax effect  
- fade‑in / fade‑out animations  
- responsive information cards  
- article section with overlay  
- elegant contact form  

### 📱 Mobile-first
- hamburger menu  
- scroll lock when menu is open  
- full responsiveness down to 480px
